# Go语言爱好者周刊

随着 Go 受到越来越多的关注，相关的文章、软件、资源也越来越多。Go语言爱好者周刊旨在为大家分享一周值得了解、学习的 Go语言相关内容。每周日发布。

欢迎投稿，推荐或自荐文章/软件/资源，请[提交 issue](https://github.com/polaris1119/golangweekly/issues) 。
